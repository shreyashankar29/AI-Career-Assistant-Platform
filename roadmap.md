## Course Modules and Weekly Breakdown

### Week 1: 1. Transformer Models
* **Core Focus:** The shift to self-attention. Deep dive into Encoder (BERT), Decoder (GPT), and Encoder-Decoder (T5) architectures.
* **Deliverable:** Architectural breakdown and performance comparison mapping.

### Week 2: 2. Using 🤗 Transformers
* **Core Focus:** Navigating the ecosystem. Loading models, leveraging pipelines for inference, and understanding AutoClasses.
* **Deliverable:** Scripting basic inference tasks across multiple model families.

### Week 3: 3. Fine-Tuning a Pretrained Model
* **Core Focus:** Domain adaptation. Utilizing the Trainer API, setting up training loops, and evaluating loss metrics.
* **Deliverable:** A fully fine-tuned text-classification model.

### Week 4: 4. Sharing Models and Tokenizers
* **Core Focus:** Version control and model registries. Pushing artifacts to the Hub and managing model cards for documentation.
* **Deliverable:** An open-sourced model with a comprehensive, professional model card.

### Week 5: 5. The 🤗 Datasets Library
* **Core Focus:** Data pipelines. Loading, mapping, filtering, and streaming massive datasets without memory bottlenecks.
* **Deliverable:** A scalable ingestion pipeline handling out-of-core datasets.

### Week 6: 6. The 🤗 Tokenizers Library
* **Core Focus:** The sub-word layer. BPE, WordPiece, and Unigram algorithms. Training custom tokenizers from scratch.
* **Deliverable:** A custom tokenizer trained on a niche, domain-specific corpus.

### Week 7: 7. Classical NLP Tasks
* **Core Focus:** Applied problem-solving. Implementing sequence classification, token classification (NER), and Extractive QA.
* **Deliverable:** A multi-task inference API handling diverse text queries.

### Week 8: 8. How to Ask for Help
* **Core Focus:** Debugging and community engagement. Crafting minimal reproducible examples and navigating ecosystem issues.
* **Deliverable:** A mock debugging sprint identifying and resolving a broken training script.

### Week 9: 9. Building and Sharing Demos
* **Core Focus:** Prototyping and UI. Wrapping models into interactive web applications using Gradio or Streamlit on Spaces.
* **Deliverable:** A live, public-facing web interface for the Week 7 multi-task model.

### Week 10: 10. Curate High-Quality Datasets
* **Core Focus:** Data engineering for AI. Scraping, cleaning, deduplication, and generating synthetic data using stronger models.
* **Deliverable:** A clean, formatted instruction-tuning dataset ready for LLM consumption.

### Week 11: 11. Fine-Tune Large Language Models
* **Core Focus:** Parameter-Efficient Fine-Tuning (PEFT). Mastering LoRA, QLoRA, and alignment techniques (RLHF/DPO).
* **Deliverable:** An instruction-tuned LLM optimized for consumer GPU hardware.

### Week 12: 12. Build Reasoning Models
* **Core Focus:** Agentic behavior. Implementing Chain-of-Thought (CoT), tool use, and multi-step orchestration.
* **Deliverable:** A reasoning agent capable of routing queries and executing external tools.
